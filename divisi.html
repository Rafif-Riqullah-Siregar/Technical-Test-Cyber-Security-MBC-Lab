Divisi

Terdapat 4 Divisi pada MBC Laboratory, yaitu :

Cyber Security

Cyber Security mencakup layanan untuk melindungi sistem digital, jaringan, dan data dari akses tidak sah, serangan siber, dan kebocoran informasi. Layanan utamanya meliputi:

  • Network Security, yaitu melindungi jaringan dari peretasan, sniffing, dan serangan DDoS.
  • Application Security, yaitu menjaga keamanan aplikasi dari bug, malware, dan eksploitasi.
  • Security Monitoring & Incident Response, yaitu deteksi ancaman dan respon cepat terhadap insiden.

Big Data

Big Data mencakup layanan pengumpulan, penyimpanan, pengolahan, analisis, dan visualisasi data dalam volume besar, baik terstruktur maupun tidak terstruktur. Layanan meliputi:

  • Data Analysis, yaitu analisis data untuk menemukan pola, tren, dan insight melalui statistik, machine learning, dan AI.
  • Data Visualization, yaitu penyajian data dalam bentuk grafik atau dashboard agar mudah dipahami (menggunakan Power BI, Tableau, dsb).
  • Data Security & Governance, yaitu menjaga integritas, privasi, dan kepatuhan data terhadap regulasi.

Global Information System (GIS)

GIS adalah sistem yang digunakan untuk mengumpulkan, menyimpan, menganalisis, dan memvisualisasikan data spasial atau geografi. Layanan utama GIS meliputi:

  • Pemetaan Digital, yaitu pembuatan dan penyajian peta interaktif berbasis data lokasi.
  • Pengelolaan Data Geospasial, yaitu penyimpanan dan pengolahan data berbasis lokasi (peta, koordinat, citra satelit).
  • Pelacakan dan Pemantauan, yaitu real-time tracking aset, kendaraan, cuaca, atau bencana.

Game Tech

Game Tech mencakup teknologi, sistem, dan layanan yang digunakan untuk mengembangkan, menjalankan, dan mengelola game digital di berbagai platform. Layanannya meliputi:

  • Game Development Tools, yaitu mesin game seperti Unity, Unreal Engine, Godot untuk membuat game 2D/3D.
  • Cloud Gaming & Backend Services, yaitu layanan server real-time, multiplayer, penyimpanan data pemain, dan live update.
  • AI & Game Analytics, yaitu penerapan kecerdasan buatan untuk NPC (Non-Player Character), personalisasi gameplay, dan analisis perilaku pemain.
